# Fantasy Hockey App

A real-time fantasy hockey application that displays live NHL game data, scores, and player statistics.

## Features

- Live game tracking with real-time score updates
- Detailed game statistics and boxscores
- Player performance tracking
- Game schedule calendar
- Team management interface
- Responsive design for mobile and desktop

## API Integration

This app uses the NHL Stats API to fetch real-time game data, scores, and player statistics. The API endpoints include:

- Game Schedule: `/api/schedule`
- Live Game Data: `/api/games/{id}`
- Player Statistics: `/api/players/{id}/stats`
